=================================

binutils 2.26<br/>
gcc 5.3.0<br/>
mpfr 3.1.3<br/>
gmp 6.1.0<br/>
mpc 1.0.3<br/>
linux 4.4.2<br/>
glibc 2.23<br/>
libstdc++ 5.3.0 (part of gcc 5.3.0)<br/>
<font color="blue">
**WARNING: Other version not tested**
</font><br/>

source library-check.sh<br/>
source version-check.sh<br/>

source build.1st.sh<br/>
